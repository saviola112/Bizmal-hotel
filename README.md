# 🏨 Bizmal Hotel Landing Page

## 📝 Project Overview

This is an interactive, responsive landing page for the Bizmal Hotel. Originally a static site, it has been updated with core JavaScript functionality to provide a dynamic user experience.

## ⚙️ Features & Functionality

I have implemented the following JavaScript features to move beyond a basic UI:

- **Dynamic Booking Calculator:** Automatically calculates the total cost of a stay based on the selected room type and number of nights.
- **Time-Based Greetings:** The homepage greets users with "Good Morning," "Afternoon," or "Evening" based on their local time.
- **Form Interaction:** Handles contact inquiries with validation and custom feedback messages.
- **DOM Manipulation:** Real-time updates to the UI without page refreshes.

## 🚀 Technologies Used

- **HTML5:** Structured semantic content.
- **CSS3:** Custom styling and responsive design using Media Queries.
- **JavaScript (ES6+):** Logic for calculations, greetings, and event handling.

## 🛠️ How to View

1. Clone the repository.
2. Open `index.html` in your web browser.
3. Use the **Booking Calculator** to see the JavaScript logic in action!
