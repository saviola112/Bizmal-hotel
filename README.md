# 🏨 Bizmal Hotel Landing Page

## Project Overview

This project is a static, responsive landing page for the Bizmal Hotel, built as an exercise in foundational front-end development.

## 🛠️ Technologies Used

* **HTML5:** Structure and content.
* **CSS3:** Styling, layout, and **responsive design** using media queries.

## 🚀 How to View

1.  Clone the repository.
2.  Open `index.html` in your web browser.
